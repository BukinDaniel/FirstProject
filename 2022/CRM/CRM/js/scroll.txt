Управление прокруткой страницы:
1.Метод scrollBy(x,y) прокручивает страницу относительно её текущего положения

function setScrollBy(){
	window.scrollBy(0, 50);
	const windowScrollTop = window.pageYOffset;
	console.log(windowScrollTop);
}

2. Метод ScrollTo(pageX, pageY) прокручивает страницу на абсолютные координаты(те, чо заданы в параметрах). Тоже самое что и window.scroll()
function setScrollTo(){
	window.scrollTo(0,50);
	const windowScrollTop = window.pageYOffset;
	console.log(windowScrollTop);
}

Метод scrollTo + опции имеет следующий синтаксис(опции не работают в сафари):
function setScrollToOptions(){
	window.scrollTo({
	top: 500,
	left: 0,
// smooth, instant, либо auto (по умолчанию auto)
	behavior : "smooth" // behavior - параметр отвечающий за тип прокрутки. smooth - плавная прокрутка.
	});
}

Метод scrollIntoView(top) прокручивает страницу, чтобы elem оказался вверху. У него есть один аргумент:
- если top = true(по умолчанию), то страница будет прокручена,чтобы elem появился в верхней части окна. Верхний край элемента совмещен с верхней частью окна.
- если top = false, то страница будет прокручена, чтобы elem появился внизу. Ниний край элемента будет ссовмещен с нижнеим краем окна.

function setScrollIntoView(top) {
	const lessonSelected = document.querySelector('/lesson_selected');
	lessonSelected.scrollIntoView(top);
}
так же у этого метода есть опции, также не работают в сафаре, единственное что, другие опции:
block: "center" //так же может быть star end или nearest отвечает за вертикальное позиционированние
inline: "center" //так же может быть star end или nearest отвечает за горизонтальное позиционированние
behavior: "smooth"
